# DSA-Leetcode

A collection of Data Structures and Algorithms (DSA) problem solutions from LeetCode.

## Problems

| # | Title | Difficulty | Solution |
|---|-------|------------|----------|
| 1 | [Two Sum](https://leetcode.com/problems/two-sum/) | Easy | [Python](./problems/001-two-sum/solution.py) |

## How to Use

Each problem is organized in its own folder under `problems/`. The folder name follows the pattern `{problem-number}-{problem-name}`.

Each solution includes:
- Problem description and examples
- Solution approach explanation
- Time and space complexity analysis
- Test cases

## Running Solutions

To run a solution and verify the test cases:

```bash
python3 problems/001-two-sum/solution.py
```
